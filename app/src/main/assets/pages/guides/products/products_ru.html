В данной инструкции будет рассказано, как перейти к информации по определению свежести для конкретного продукта.

Чтобы получить информацию по определению свежести для конкретного продукта в главном меню выберите пункт "Продукты". В открывшемся экране отображается список доступных продуктов (смотрите скриншот ниже).

Выберите необходимый Вам продукт, тем самым Вы перейдете на страницу с информацией по определению свежести для выбранного продукта.
