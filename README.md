# Распознователь свежести продуктов питания

### Описание проекта
Android приложение для распознавания свежести продукта питания по фотографии с камеры телефона.

### Основной стек технологий:
* Android
* Java
* Tensorflow
* Git.

## СРЕДА ЗАПУСКА
Требуется установленный Android Studio.
